# N Sparkle - Task Tracker

A beautiful, modern, and fully-featured task management application with multiple theme options and comprehensive productivity tools.

## ✨ Features

- **Task Management**: Create, edit, and organize your tasks with ease
- **Priority Levels**: Set task priorities (Low, Medium, High) with color-coded indicators
- **Multiple Themes**: 
  - Light Mode
  - Dark Mode
  - Pastel Mode
- **Habit Tracking**: Track daily habits and build streaks
- **Statistics Dashboard**: View your productivity metrics at a glance
- **Task Completion Tracking**: Mark tasks as complete and visualize progress
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Beautiful UI**: Built with pastel colors and modern glassmorphism effects

## 🛠️ Technologies Used

- **HTML5**: Semantic markup structure
- **Bootstrap 5**: Responsive grid and components
- **Tailwind CSS**: Utility-first CSS framework
- **Font Awesome**: Icon library
- **Google Fonts**: Custom typography (Poppins & Quicksand)
- **JavaScript**: Interactive functionality and state management

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No installation required!

### Usage

1. **Open the Application**
   - Simply open `TODOLIST.html` in your web browser

2. **Create a Task**
   - Click the "Add Task" button
   - Enter your task details
   - Select a priority level
   - Click save

3. **Manage Tasks**
   - Mark tasks as complete by clicking the checkbox
   - Delete tasks using the delete button
   - Filter tasks by priority or status

4. **Switch Themes**
   - Click the theme selector buttons in the header
   - Choose between Light, Dark, or Pastel modes
   - Your preference is automatically applied

5. **Track Progress**
   - View your statistics in the dashboard
   - Monitor habit completion rates
   - Check daily streaks

}
